# Homepage Image Prompts

## Global Style Guidelines

All images must follow these specifications:

- **Style**: Flat vector graphics
- **Background**: Light (#ffffff or #f7f9fb)
- **Colors**: Slate greys (#334155, #64748b) with Taubyte green (#00c853) accents only
- **Strokes**: Uniform 2px
- **Corners**: 4px radius
- **Forbidden**: Gradients, shadows, blur effects
- **Forbidden**: Text, letters, numbers, logos, watermarks, or UI chrome
- **Output**: Single variation preferred
- **Format**: SVG preferred, or PNG with minimum 2048px on the long edge
